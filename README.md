ImagenesSudoku
Este repositorio contiene las herramientas de software desarrolladas para la generación de la base de datos visual y estructural del proyecto de investigación sobre la Geometría del Sudoku.

El objetivo principal de este módulo es procesar las plantillas (templates) y grafos de compatibilidad para generar representaciones en formato de imagen y archivos de datos planos que permitan el análisis posterior de la estructura del Sudoku como un objeto geométrico de alta dimensión.

🚀 Descripción del Proyecto
A diferencia de los enfoques tradicionales que ven el Sudoku como un simple rompecabezas lógico, este proyecto lo aborda desde la Teoría de Grafos y la Topología. Este repositorio específicamente se encarga de:

Generación de Templates: Creación de las estructuras base para las 9 filas, 9 columnas y 9 regiones.

Exportación de Datos: Generación de archivos .txt que contienen las matrices de adyacencia y compatibilidad.

Renderizado Visual: Creación de representaciones gráficas de las celdas y sus interconexiones.

🛠️ Tecnologías Utilizadas
Lenguaje: Visual Basic .NET / VB6 (Núcleo de procesamiento lógico).

Salida de Datos: Formatos de imagen estándar y archivos de texto plano para integración con herramientas de visualización avanzada.

📁 Estructura del Repositorio
/src: Código fuente de los algoritmos de generación.

/Ejemplos: Muestras de las imágenes generadas y los archivos de datos resultantes.

/Docs: Documentación técnica sobre la lógica de las plantillas de Sudoku.

📐 Contexto Académico
Este software es una pieza fundamental para la generación de evidencias en el paper:

"A Structural Characterization of Sudoku via Templates and Compatibility Graphs"

Donde se demuestra que un Sudoku 9x9 puede ser mapeado como un subgrafo específico dentro de una estructura geométrica compleja (hexeracto).
